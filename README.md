# Лабораторна робота №3 з Python

**Виконав:** Качоровський Данііл (КІб-1-24-4.0д)

## Опис

Цей репозиторій містить виконання лабораторної роботи №3 на тему **«Реалізація обробки виключень, модульності та документації в проєктах Python»**.

Проєкт реалізовано у вигляді пакета `inventory_app`, який демонструє використання механізмів обробки виключень, створення власних класів виключень, модульної структури Python-проєкту та документування програмного коду.

### Структура проєкту

- `task1.py` — Демонстрація обробки виключень за допомогою конструкцій `try/except`, `try/except/else/finally`, а також перехоплення кількох типів виключень одним блоком `except`.
- `task2.py` — Реалізація ієрархії власних виключень, використання ланцюжків виключень через `raise ... from ...` та повторного піднесення виключень.
- `task3.py` — Демонстрація модульної структури проєкту, роботи пакетів, імпортів та механізму `__all__`.
- `task4.py` — Демонстрація документування коду за допомогою docstrings, анотацій типів та інструментів інтроспекції (`help()`, `__doc__`, `__name__`, `__annotations__`).

### Пакет inventory_app

- `models/exceptions.py` — власні класи виключень системи.
- `models/inventory.py` — клас для роботи зі складом товарів.
- `handlers/command_handler.py` — функції обробки команд та демонстрація ланцюжків виключень.
- `utils/logger.py` — журналювання повідомлень.
- `utils/formatter.py` — форматування даних для виведення.
- `__main__.py` — точка входу для запуску всього проєкту.
